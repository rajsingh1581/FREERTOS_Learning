# FREERTOS_Learning
This repository contains code of learning in Free RTOS From Fastbit lectures.

This branch contains of Nucleo-L476RG Board materials
This board have enabled following featurs
1) UART
  a) UART1 - PA10 - RX
			 PA9  - TX
  b) UART2 - PA3  - RX
			 PA2  - TX

2) USER LED 
             PA5  - LD2
3) Button 
			 PC13 - Button

4) ADC
			 PC0  - ADC1_IN1
